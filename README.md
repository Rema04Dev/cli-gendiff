### Description
A simple utility that compares two configuration files (json, yml/yaml)

### Setup
1. Clone this repository
```
    git clone git@github.com:Rema04Dev/frontend-project-lvl2.git
```
2. Install dependenses
```
make install
```
3. Link the package
```
npm link
```
